Loops

This module covers loop constructs for repeating actions in Bash scripts.

for

Description: Repeats a block of code for each item in a list.

Syntax: for var in list; do ...; done

Example: for file in *.txt; do echo $file; done

Tip: Use seq for numeric ranges: for i in $(seq 1 5); do ...; done

while

Description: Repeats a block of code while a condition is true.

Syntax: while [ condition ]; do ...; done

Example: while [ $count -lt 5 ]; do echo $count; ((count++)); done

Tip: Useful for reading files line by line: while read line; do ...; done < file.txt

until

Description: Repeats a block of code until a condition becomes true.

Syntax: until [ condition ]; do ...; done

Example: until [ $done -eq 1 ]; do echo "Working"; done=1; done

Tip: Like while, but runs until the condition is true.

break

Description: Exits the nearest enclosing loop.

Syntax: break

Example: for i in {1..10}; do if [ $i -eq 5 ]; then break; fi; done

Tip: Use to stop a loop early based on a condition.

continue

Description: Skips the rest of the current loop iteration and continues with the next.

Syntax: continue

Example: for i in {1..5}; do if [ $i -eq 3 ]; then continue; fi; echo $i; done

Tip: Use to skip processing for certain values in a loop.
