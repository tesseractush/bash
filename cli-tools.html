CLI Tools

This module covers advanced command-line interface tools in Bash.

getopts (option parsing)

Description: Parses command-line options in scripts.

Syntax: while getopts "ab:c" opt; do ... done

Example: while getopts "f:v" opt; do case $opt in f) file=$OPTARG;; v) verbose=1;; esac; done

Tip: Use getopts for flexible script interfaces.

select (interactive menus)

Description: Creates simple interactive menus in scripts.

Syntax: select var in list; do ...; done

Example: select fruit in apple banana cherry; do echo $fruit; break; done

Tip: Use break to exit the menu after a selection.
