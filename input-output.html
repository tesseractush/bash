Input/Output Control

This module covers advanced input and output techniques in Bash.

Here-docs / Here-strings

Description: Multi-line input for commands or scripts.

Syntax: command <<EOF ... EOF (here-doc), command <<< "string" (here-string)

Example: cat <<EOF\nHello\nWorld\nEOF

Tip: Use here-docs for script templates or config files.

Process substitution (<(cmd))

Description: Treats command output as a file input.

Syntax: command <(other_command)

Example: diff <(ls dir1) <(ls dir2)

Tip: Useful for comparing outputs or feeding input to commands that expect files.
