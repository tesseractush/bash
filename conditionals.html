Conditionals

This module covers conditional logic in Bash scripts.

if / elif / else

Description: Executes code blocks based on conditions.

Syntax: if [ condition ]; then ... elif [ condition ]; then ... else ... fi

Example: if [ $age -ge 18 ]; then echo "Adult"; else echo "Minor"; fi

Tip: Always close with fi. Use elif for multiple branches.

Test operators ([ ], [[ ]])

Description: Used to evaluate conditions in Bash.

Syntax: [ condition ] or [[ condition ]]

Example: [ -f file.txt ] (checks if file exists)

Tip: [[ ]] is more flexible and supports pattern matching.

case

Description: Multi-way branch statement for matching patterns.

Syntax: case $var in pattern1) ... ;; pattern2) ... ;; *) ... ;; esac

Example: case $color in red) echo "Stop";; green) echo "Go";; *) echo "Unknown";; esac

Tip: Always end with esac. Patterns can use wildcards.
