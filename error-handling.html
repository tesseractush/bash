Error Handling

This module covers robust error handling in Bash scripts.

set -e, set -u, set -o pipefail

Description: Options to make scripts fail on errors, unset variables, or failed pipes.

Syntax: set -e | set -u | set -o pipefail

Example: set -euo pipefail

Tip: Use at the top of scripts for safety.

Custom error messages

Description: Print helpful messages when errors occur.

Syntax: echo "Error: ..." 1>&2

Example: if [ ! -f file ]; then echo "File missing" 1>&2; exit 1; fi

Tip: Send errors to stderr with 1>&2.

Trap signals (trap 'cleanup' EXIT)

Description: Run a command or function when the script exits or receives a signal.

Syntax: trap 'commands' SIGNAL

Example: trap 'echo Goodbye' EXIT

Tip: Use for cleanup tasks or logging.
