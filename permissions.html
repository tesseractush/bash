Permissions

This module covers commands for managing file permissions and ownership.

chmod

Description: Changes file or directory permissions.

Syntax: chmod [options] mode file

Example: chmod +x script.sh

Tip: Use chmod 755 file for common executable permissions.

sudo

Description: Executes a command as another user (usually root).

Syntax: sudo [command]

Example: sudo apt update

Tip: Use sudo !! to rerun the last command with sudo.

chown

Description: Changes file owner and group.

Syntax: chown [owner][:group] file

Example: chown user:group file.txt

Tip: Use -R for recursive ownership changes on directories.
