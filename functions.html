Functions

This module covers defining and using functions in Bash scripts.

Definition and invocation

Description: Functions group reusable code blocks.

Syntax: function name() { commands; } or name() { commands; }

Example: greet() { echo "Hello, $1"; }\ngreet Alice

Tip: Define functions at the top of your script for clarity.

Local vs global scope

Description: Variables in functions are global unless declared local.

Syntax: local var=value

Example: myfunc() { local x=5; echo $x; }

Tip: Use local to avoid variable conflicts.

Return values

Description: Functions return exit codes (0-255) with return, or output with echo.

Syntax: return [code] or echo [value]

Example: add() { echo $(($1 + $2)); }\nresult=$(add 2 3)

Tip: Use echo for data, return for status.
