Arrays & Associative Arrays

This module covers indexed and associative arrays in Bash.

Indexed arrays

Description: Arrays with integer indices.

Syntax: arr=(val1 val2 val3)

Example: arr=(apple banana cherry); echo ${arr[1]}

Tip: Use ${arr[@]} to access all elements.

Associative arrays (Bash 4+)

Description: Arrays with string keys (key-value pairs).

Syntax: declare -A map; map[key]=value

Example: declare -A colors; colors[apple]=red; echo ${colors[apple]}

Tip: Always declare with declare -A before use.
