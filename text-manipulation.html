Text Manipulation

This module covers commands for viewing, searching, and manipulating text files and streams.

cat

Description: Concatenates and displays file contents.

Syntax: cat [file]

Example: cat notes.txt

Tip: Use cat file1 file2 to view multiple files together.

head

Description: Displays the first lines of a file.

Syntax: head [options] [file]

Example: head -n 5 data.txt

Tip: Default is 10 lines. Use -n to specify a different number.

tail

Description: Displays the last lines of a file.

Syntax: tail [options] [file]

Example: tail -f /var/log/syslog

Tip: -f follows the file as it grows (useful for logs).

grep

Description: Searches for patterns in files or input.

Syntax: grep [pattern] [file]

Example: grep error logfile.txt

Tip: Use -i for case-insensitive search, -r for recursive.

cut

Description: Removes sections from each line of files.

Syntax: cut -d [delimiter] -f [fields] [file]

Example: cut -d "," -f 1,3 data.csv

Tip: Useful for extracting columns from CSV files.

sort

Description: Sorts lines of text files.

Syntax: sort [options] [file]

Example: sort names.txt

Tip: Use -r for reverse order, -n for numeric sort.

> and >>

Description: Redirects output to a file. > overwrites, >> appends.

Syntax: command > file or command >> file

Example: echo Hello > hello.txt

Tip: Use >> to add to a file without erasing its contents.

| (pipe)

Description: Sends the output of one command as input to another.

Syntax: command1 | command2

Example: ls | grep txt

Tip: Pipes are powerful for chaining commands together.
