Shell Basics

This module covers the essential commands to navigate and interact with the shell.

ls

Description: Lists directory contents.

Syntax: ls [options] [directory]

Example: ls -l /home/user

Tip: Use ls -a to show hidden files. Combine options, e.g., ls -la.

cd

Description: Changes the current working directory.

Syntax: cd [directory]

Example: cd /var/log

Tip: cd ~ takes you to your home directory. cd - switches to the previous directory.

pwd

Description: Prints the current working directory.

Syntax: pwd

Example: pwd

Tip: Useful to confirm your location after using cd.

man

Description: Displays the manual page for a command.

Syntax: man [command]

Example: man ls

Tip: Press q to quit the manual. Use / to search within the manual page.
