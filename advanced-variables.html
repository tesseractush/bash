Advanced Variables

This module covers advanced variable usage in Bash scripts.

Default values (${VAR:-default})

Description: Uses a default value if a variable is unset or empty.

Syntax: ${VAR:-default}

Example: echo "Hello, ${name:-Guest}"

Tip: Useful for scripts that accept optional arguments.

String operations (${var//old/new})

Description: Replaces all occurrences of a substring in a variable.

Syntax: ${var//old/new}

Example: text="foo bar foo"; echo ${text//foo/baz}

Tip: Use ${var/old/new} to replace only the first occurrence.
