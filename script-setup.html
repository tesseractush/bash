Script Setup

This module covers the basics of writing and running Bash scripts.

Shebang (#!/bin/bash)

Description: The first line in a script that tells the system which interpreter to use.

Syntax: #!/bin/bash

Example: #!/bin/bash\necho Hello

Tip: Use #!/usr/bin/env bash for portability across systems.

Variables

Description: Store values for use in scripts.

Syntax: VAR=value

Example: name="Alice"

Tip: No spaces around =. Access with $VAR.

echo

Description: Prints text to the terminal.

Syntax: echo [text]

Example: echo "Hello, $name"

Tip: Use quotes to preserve spaces and special characters.

read

Description: Reads user input into a variable.

Syntax: read [variable]

Example: read username

Tip: Use -p to prompt: read -p "Enter name: " name

Making scripts executable

Description: Allows a script to be run directly from the terminal.

Syntax: chmod +x script.sh

Example: chmod +x myscript.sh

Tip: Run with ./myscript.sh after making it executable.
